🚀 快速开始 (8 分钟)
================================================================================

1. 启动服务 (2分钟)
   docker-compose up -d

2. 安装依赖 (3分钟)
   cd backend && npm install
   cd ../frontend && npm install

3. 配置 API (1分钟)
   cp backend/.env.example backend/.env
   # 编辑 .env，添加 OPENAI_API_KEY

4. 启动应用 (2分钟)
   # 终端 1: 后端
   cd backend && npm run start:dev

   # 终端 2: 前端
   cd frontend && npm run dev

5. 访问应用
   http://localhost:5173
