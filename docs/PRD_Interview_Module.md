## 1. 产品概述

### 1.1 产品背景

当前系统已实现用户认证、笔记管理、简历分析、知识库、AI助手等核心功能。模拟面试模块作为系统的最后一个核心功能模块，旨在为求职者提供AI驱动的面试模拟体验，帮助用户：

- 熟悉面试流程，降低面试焦虑
- 针对特定岗位进行定向练习
- 获得实时的回答评估和改进建议
- 积累面试经验，提升面试表现

### 1.2 产品定位

模拟面试模块定位为**AI驱动的智能面试训练平台**，通过模拟真实面试场景，结合用户简历和知识库内容，提供个性化的面试体验和专业反馈。

### 1.3 产品目标

| 目标类型 | 具体目标 | 衡量指标 |
|---------|---------|---------|
| 用户价值 | 帮助用户提升面试表现 | 用户完成面试后满意度≥4.0/5.0 |
| 功能完善 | 补全系统核心功能模块 | 模块上线后系统功能完整度达100% |
| 用户留存 | 提升用户粘性和活跃度 | 模块上线后DAU提升≥20% |
| 差异化竞争 | 打造产品核心竞争力 | 面试模拟功能用户使用率≥60% |

### 3.2 功能模块详细说明

#### 3.2.1 面试形式选择（P0）

**功能描述：**
用户在开始面试前选择面试形式，系统根据选择加载相应的面试界面和功能模块。

**面试形式类型：**

| 面试形式 | 形式代码 | 描述 | 适用场景 |
|---------|---------|------|---------|
| 文字对话 | text | 纯文字交互，支持语音输入 | 基础练习、快速模拟 |
| 语音通话 | voice | 纯语音交互，无视频 | 电话面试模拟、通勤练习 |
| 视频面试 | video | 视频通话，含虚拟面试官 | 真实面试模拟、综合训练 |

#### 3.2.2 面试对话（P0）

**功能描述：**
用户与AI面试官进行实时对话，AI根据场景和用户回答动态生成问题和追问。

**对话流程：**

```
┌─────────────────────────────────────────────────────────────────┐
│                        面试对话流程                              │
└─────────────────────────────────────────────────────────────────┘

AI面试官                              用户
    │                                   │
    │  1.开场白+第一个问题              │
    ├──────────────────────────────────▶│
    │                                   │
    │                                   │  2.思考并回答
    │                                   ├──────────────┐
    │                                   │              │
    │                                   │◀─────────────┘
    │  3.评估回答+追问或下一题          │
    ├──────────────────────────────────▶│
    │                                   │
    │              ... 重复N轮 ...      │
    │                                   │
    │  N.结束语+面试结束提示            │
    ├──────────────────────────────────▶│
    │                                   │
    │                                   │  查看面试报告
    │                                   ├──────────────┐
    │                                   │              │
    │                                   │◀─────────────┘
    │                                   │
```

**问题生成策略：**

| 阶段 | 问题数量 | 问题类型 | 生成策略 |
|------|---------|---------|---------|
| 开场 | 1 | 自我介绍 | 固定模板 |
| 主体 | 5-8 | 核心问题 | 场景+岗位+难度组合 |
| 追问 | 2-4 | 深度挖掘 | 基于用户回答动态生成 |
| 结束 | 1 | 反向提问 | 固定模板 |

**AI面试官角色设定：**

```
角色：专业面试官
- 语气：专业、友善、适度压力
- 行为：认真倾听、适时追问、给予鼓励
- 评估维度：回答完整性、逻辑性、专业性、表达能力
```

**界面元素：**
- AI面试官头像和名称
- 消息列表（支持Markdown渲染）
- 用户输入框（支持多行）
- 发送按钮
- 结束面试按钮
- 当前进度指示器（第X题/共Y题）
- 计时器（可选显示）

#### 3.2.3 文字对话面试增强 - 语音输入（P1）

**功能描述：**
在现有文字对话基础上，增加语音输入功能，允许候选人通过语音方式输入回答内容，系统需将语音准确转换为文字并显示在对话界面中，主体交互形式仍保持文字对话模式。

**技术方案：**

```typescript
// interview.entity.ts
@Entity('interviews')
export class Interview {
  @PrimaryGeneratedColumn('uuid')
  id!: string;

  @Column({ type: 'varchar', length: 100 })
  userId!: string;

  @Column({ type: 'varchar', length: 50 })
  sceneType!: string;

  @Column({ type: 'varchar', length: 50, nullable: true })
  jobType?: string;

  @Column({ type: 'varchar', length: 20, default: 'medium' })
  difficulty!: string;

  @Column({ type: 'uuid', nullable: true })
  resumeId?: string;

  @Column({ type: 'float', nullable: true })
  totalScore?: number;

  @Column({ type: 'integer', nullable: true })
  duration?: number;

  @Column({ 
    type: 'varchar', 
    length: 20, 
    default: 'in_progress' 
  })
  status!: string;

  @CreateDateColumn()
  createdAt!: Date;

  @UpdateDateColumn()
  updatedAt!: Date;

  @ManyToOne(() => User, (user) => user.interviews)
  user!: User;

  @ManyToOne(() => Resume, { nullable: true })
  resume?: Resume;

  @OneToMany(() => InterviewSession, (session) => session.interview)
  sessions!: InterviewSession[];

  @OneToOne(() => InterviewReport, (report) => report.interview)
  report?: InterviewReport;
}
```

### 8.4 流式响应实现

```typescript
// interview-llm.service.ts
async streamChat(
  sessionId: string,
  message: string,
  onChunk: (chunk: string) => void,
  onComplete: (evaluation: EvaluationResult) => void,
): Promise<void> {
  const messages = await this.getMessageHistory(sessionId);
  messages.push({ role: 'user', content: message });

  const stream = await this.openai.chat.completions.create({
    model: 'gpt-4',
    messages,
    stream: true,
  });

  let fullContent = '';
  for await (const chunk of stream) {
    const content = chunk.choices[0]?.delta?.content || '';
    fullContent += content;
    onChunk(content);
  }

  const evaluation = await this.evaluateAnswer(message, fullContent);
  onComplete(evaluation);
}
```
